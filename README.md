# MUD 2.0 Web3 dApp Game Demo

Demo application demonstrating various uses of store, systems and querying ECS data with recs.

Built with:
- Solidity
- MUD 2.0
- React
- Tailwind

## Run Application

Build and execute:
```
pnpm dev
```

Navigate to:  http://localhost:3000/
